单调栈

单调栈具有如下性质：

  • 若是单调递增栈，则从栈顶到栈底的元素是严格递增的；若是单调递减栈，则从栈顶到栈底的元素是严格递减的。
  • 对于单调递增栈，若当前进栈元素比栈顶元素大(或相等)，则应将栈顶元素出栈，直至进栈元素比栈顶元素小；单调递减栈同理。
  • 单调递增栈栈顶元素左边保存的是数组左边第一个比栈顶元素大的元素。

1.右侧第1个比自己大的数

给一个数组，返回一个大小相同的数组。返回的数组的第i个位置的值应当是，对于原数组中的第i个元素，至少往右走多少步，才能遇到一个比自己大的元素（如果之后没有比自己大的元素，或者已经是最后一个元素，则在返回数组的对应位置放上-1）。
示例：
input:[5 3 1 2 4]
output:[-1 3 1 1 -1]

分析

维护一个单调递增栈，栈内保存数组元素下标，对于一个进栈元素，若比当前栈顶下标对应的元素大，说明该元素是比栈顶下标对应元素大的第一个数，则对应位置结果为二者下标差，将栈顶元素出栈，继续比较进栈元素和进栈元素大小，直至栈顶元素比进栈元素大，说明栈顶元素之前的元素都比进栈元素大。

2.Largest Rectangle in Histogram

分析

和上题差不多，对于每个bar，找到右边第一个比它矮的bar,则它能围成的矩面积形为它的高乘以第一个比它矮的bar的下标和左边第一个比它矮的下标差。如果右边没有比它矮的bar,则能围成的面积为它的高乘以数组大小减去左边第一个比它矮的下标，如果左边没有比它矮的，下标即为-1,。也可以在数组末尾添加1个0，使得右边一定有个比任何bar都矮的。

3.Maximal Rectangle

分析

这道题可以转化为上一道题，把每一列看做一个bar,对于每行元素，如果某元素为1，则其高度为上一行对应列元素高度加1，否则该元素高度为0。这样每一行我们都可以构造一个bar的高度数组heights,把heights传入上一题函数中，可得本行最大矩形面积；遍历每一行，即可得到答案。
对于示例：
第一行，对应的高度为 1 0 1 0 0
第二行，对应的高度为 2 0 2 1 1
第三行，对应的高度为 3 1 3 2 2
第四行，对应的高度为 4 0 0 3 0

单调队列

单调队列是一个双端的队列，也就是通常说的Deque，它既可以从队列的头取走元素，也可以从队列的尾取走元素。一般用于动态规划中或者用于动态维护子数组的最值（最小值或者最大值）。

1.Sliding Window Maximum

分析

我们用一个单调队列维护窗口内的最大值的下标，对于每个进队元素，如果当前元素比队尾元素大，队尾元素不可能是窗口内最大值，则将队尾元素出队，直至找到一个不小于当前元素的队尾元素，或者队列为空；
