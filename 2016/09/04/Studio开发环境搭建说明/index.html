Intel Parallel Studio开发环境搭建说明

安装Intel Parallel Studio

  1. 安装visual studio 2013

  2. 去英特尔官网使用学校邮箱注册账号(https://software.intel.com/registration/?lang=en-us)

  3. 使用学校邮箱申请免费使用Parallel Studio(https://registrationcenter.intel.com/en/forms/?licensetype=2&productid=2489)
    选中所有复选框点Accept
    填写邮箱和地区点Submit
    获得序列号

  4. 下好安装文件后按照提示安装即可

如何使用ICC和MKL

  1. 打开Visual Studio新建项目

  2. 点击上方菜单栏 项目-Intel Compiler-Use Visual C++即可切换至ICC编译器

  3. 使用MKL：点击菜单栏 项目-属性-配置属性，如图启用MKL库即可

  4. 在文件中添加#include<mkl.h>即可

至此完成开发环境搭建，调用MKL完成后续模块即可
