ubuntu下环境配置

1.下载python库和头文件：打开终端执行sudo apt-get install python-dev即可。安装完成后头文件 在 /usr/include/python2.7 下，库文件在/usr/lib/python2.7/config下。

2.安装matplotlib：打开终端执行pip install matplotlib即可
